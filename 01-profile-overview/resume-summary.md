# Resume-Based Interview Questions & Answers

## Q1. Can you explain your role as an Automation Test Engineer?

**Short Interview Answer:**
I have around 2 years of experience as an Automation Test Engineer working on web-based enterprise applications. My role includes designing test cases, automating E2E scenarios using Python, Playwright, and pytest, performing API and database validation, and collaborating with development teams in Agile environments.

---

## Q2. What kind of application did you work on?

**Answer:**
I worked on an Electronic Toll Collection platform (QBOS), which manages customer accounts, vehicle and tag management, billing, and payment processing. The system requires strict validation across UI, API, and database layers.

---

## Q3. What is your main automation tech stack?

**Answer:**
My primary automation stack includes Python, Playwright for browser automation, pytest as the test runner, REST API testing using pytest/Postman, and MongoDB for backend data validation.

---

## Q4. What type of testing have you performed?

**Answer:**
I have performed functional testing, regression testing, smoke testing, system testing, accessibility testing, and end-to-end automation testing.
